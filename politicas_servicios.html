Politacas de privacidad

Finalidad del uso de datos

El uso de los datos proporcionado por el usuario, son meramete para fines de contacto, y se hace ajeno el uso para fines politico o de mal uso.

Cesión o transmisión

Los datos del usuario del servicio que han sido proporcionados no se comparten con tercero, el de uso propio de la empresa

Responsabilidades del usuario

El usuario no puede hacer uso propio del servicio y del contenido para usos propios de divulgacion o apropiacion misma

Forma de contacto

Asi mismo, si el usuario tiene algun comentario, respecto al servicio/producto proporcionado, se dejan algunos correos como forma de contacto, para aclarar dudas o sugerencias
